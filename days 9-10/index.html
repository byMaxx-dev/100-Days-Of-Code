HTML & CSS Basics Summary

HTML Summary

HTML (HyperText Markup Language) is used to define our page content, structure and meaning.
You dont use it for styling purposes. CSS for that instead!

  • HTML uses "elements" to describe (annolate) content

  • HTML elements typically have an opening tag, content and then a closing tag

  • You can also have void(empty) elements like images

  • You can also configure elements with attributes

  • There's a long list of available elements but you'll gain expereince over time, no worries

Learn more about all avaible HTML elements on the MDN HTML element reference
